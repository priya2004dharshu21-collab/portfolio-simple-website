## 🌐 Portfolio Website

This is a simple personal portfolio website built using **HTML** and **CSS**. It consists of two main pages—**Home** and **Contact**—designed to present basic information and provide an easy way to get in touch.

### ✨ Features

* Clean and minimal design
* Two-page structure (Home & Contact)
* Easy navigation between pages
* Responsive layout for different screen sizes

### 🏠 Pages Included

* **Home Page** – Introduction and overview
* **Contact Page** – Basic contact information or form

### 🛠️ Technologies Used

* HTML5
* CSS3

###  Purpose

This project was created to practice fundamental web development skills and to build a basic personal portfolio website.
